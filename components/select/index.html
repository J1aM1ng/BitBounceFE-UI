select 选择框 #

TIP

当选项过多时，使用下拉菜单展示并选择内容。

基本使用 #

传递 options 对象，name 为显示内容，value 为实际输入值。v-model 绑定选择的值。

msg:
<template>
  <div>
    <b-select :options="options" v-model="msg"></b-select>
    msg:{{ msg }}
  </div>
</template>

<script>
export default {
  name: 'selectBox',
  data() {
    return {
      options: [
        {
          name: '第一个数字值',
          value: '1234444'
        },
        {
          name: '第二个数字值',
          value: '453336'
        }
      ],
      msg: ''
    };
  }
};
</script>
